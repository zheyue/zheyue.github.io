AI Consumer Trend Dashboard

This project visualizes the popularity trends of AI productivity tools from 2024 to 2025 using data extracted from Google Trends.

📈 Dashboard Link:

View the Live Dashboard

📂 Data Files:

  • multiTimeline.csv
  • trend_long.csv

🛠️ How to Use:

Data files are available for download and can be reprocessed locally using the provided Python scripts.

📝 Insights:

  • ChatGPT for Work maintains top attention across the year.
  • Notion AI shows spikes post-feature release.
  • Generic terms like "AI productivity tools" are less engaging.
